Static and Dynamic Memory

Easy to understand how Static Memory works. It dies when it is out of scope


            int main(){
                int x = 10;
                if(true){
                    int y = 20;
                } //y dies here because it is out of scope
            } //x dies here because it is out of scope

Hard to understand how Dynamic Memory works. You need to manually create it and manually destroy it.


            int main(){
                int * x;
                int * y;
                x = new int [3];
                delete [] x; //delete dynamic array
                x = nullptr; //good practice to deallocate x

                y = new int;
                delete y; //delete dynamic single object
                y = nullptr; //good practice to deallocate y
                return 0;
            }

Declarations VS Definitions

Understanding difference between Declarations and Definitions, the simple concept is Declarations doesn't gives out information on what to do about the "thing", where Definitions provides all the details to the "thing"


            //Declarations
            int sum(int a, int b);
            int x;

            //Definitions
            int sum(int a, int b){
                return a + b;
            }
