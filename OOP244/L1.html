3 Important Concept for Object-oriented programming language

1. Encapsulation

The cout is storing the object data in memory and also controlling how it flows from the data

<< sign work as a wrapper, it encapsulate the information to the cout without exposing any sort of implementation details

Consider the exampleðŸ”°

cout << "Printing Statement";

Does: To make sure client only sees the simple output to the object without showing any further internal data

2. Inheritance

Inheritance appears when there are two class A and B, where class A is a domain class and class B is a derived class.

3. Polymorphism

Polymorphism appears when the function or methods that needs to change at compile-time(overloading) or run-time(overriding)

Always starts with compiling the code first, entering the compilation phase to check if have any uninitialized var. Next starts the execution of the program, which find some not allow procedures or undefined result.
