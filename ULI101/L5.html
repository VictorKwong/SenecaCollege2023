Redirection symbols

1 - 2 - 2 symbols, 1 standard input (stdin) | 2 standard output (stdout) | 2 standard error (stderr)


            Example ðŸ”°
            
            stdin <

            tr 'a-z' 'A-Z' < text.txt

            *text.txt has content "abc"
            *the print screen result will be "ABC" (translate all characters from lower case to upper case)
            *After running the command, text.txt remains the same content "abc"

            - Extract a copy content of text.txt, run the command tr 'a-z' 'A-Z' and print out to the screen
            - *The content of text.txt will not be modified. (stays the same content)

            -------------------------------------------------------------------

            stdout > , >>

            pwd > savePrintResult.txt

            - Save the pwd command result, and overwrite it to savePrintResult.txt
            - The file will have the following content (correct):
            /home/userid/
            - Instead of (incorrect):
            pwd

            pwd >> savePrintResult.txt

            - Save the pwd command result, and append it to savePrintResult.txt
            - The file will have the following content (correct):
            (original content)
            /home/userid/

            -------------------------------------------------------------------

            stderr 2> , 2>>

            PWD 2> saveErrorResult.txt
            
            - Save the error message, and overwrite it to saveErrorResult.txt
            - In this example, PWD has the following error message, and it will overwrite to saveErrorResult.txt:
            -bash: PWD: command not found
            -*If there is no error message, saveErrorResult.txt will have an empty content.

            PWD 2>> saveErrorResult.txt
            - It works the same as 2>, the only difference is changing overwrite to appending the saveErrorResult.txt

Black hole - /dev/null

It can dump all the unwanted information to the black hole. Once the information save into the black hole, it is gone for good.


            Example ðŸ”°

            pwd > /dev/null
            - store the pwd command result to black hole, in this case the stored information is gone for good.
