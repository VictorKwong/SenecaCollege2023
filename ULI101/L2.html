Text editor (vi/vim)

vi/vim a.txt

To enter insert mode, press i (displaying INSERT in bottom left corner of the screen)

To exit insert mode, press esc

To save and exit the file, press :x while you are not in any mode

To get more help information, press :help to get while you are not in any mode

Text editor (nano)

nano a.txt

You are already in the insert mode once you type the command nano a.txt, you can insert your content right away

When you finish and want to save it, use keyboard fast key "ctrl + x"

There will be 2 questions before saving it.

1. Want to save ? (y = yes, n = no, ctrl + c = cancel)

2. Want to change the file name? (press enter as a default file name, in this example = a.txt)

To get more help information, use keyboard fast key "ctrl + g"
