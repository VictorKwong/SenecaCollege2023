            =   8   * 1 + 4   * 1 + 2   * 0 + 1   * 0

            =   32  * 0 + 16  * 0 + 8   * 1 + 4   * 1 + 2   * 0 + 2   * 0
            =   12

File Permission

To check file permission, use command ls -l

-rwxr-xr-x if it is a file

drwxr-xr-x if it is a directory


            Example ðŸ”°

            using ls -l we have a file named a.txt with the permission shown below:
            -rwxr-xr--

            Breaking down into 3 sets of (---)
            1.First set rwx is represent to user
            2.Second set r-x is represent to groups
            3.Third set r-- is represent to others

Using chmod command to change the file permission


            Example ðŸ”°

            ls -l shows a.txt status:
            -rwxr-xr-- a.txt

            chmod 777 a.txt //Change a.txt permission to rwx rwx rwx (111 111 111 Octal = 4+2+1 = 7)

            ls -l update:
            -rwxrwxrwx a.txt //All user, group and others can edit this file

            chmod u=rwx,g=rx,o=r a.txt //Change a.txt permission to user = read write execute, group = read and execute, other = read

umask default value = 022, umask value can only works with 0,2,7

default values for directories (777) | regular files (666)


            Example ðŸ”°
            a.txt - regular files
            666 Original number
           -022 umask
            644 Octal number Result

            convert to read write and execute

            110 100 100
            rw- r-- r--

            Therefore a.txt file permission result will be rw-r--r--


            ABC - directory
            777 Original number
           -022 umask
            755 Octal number Result

            convert to read write and execute

            111 101 101
            rwx r-x r-x

            Therefore ABC file permission result will be rwxr-xr-x
